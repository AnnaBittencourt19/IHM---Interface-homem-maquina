### DOM (Document Object Model)
- HTML (Estrutura) -> DOM (localização) -> JavaScript (Comportamento)
- Document é a página HTML inteira
- Permite que um documento HTML seja acessado pelo HTML
- O DOM é como se fosse a árvore genealógica do HTML, a tag html é o patriarca, head e body os filhos e o h1 dentro do body é um neto
- A manipulação com o DOM inclui 3 etapas que são encontrar, ouvir e reagir: 
	- Encontrar: 
		- Nos permite ler elementos do documento HTML
		- É a mesma coisa de dizer "JavaScript encontre o botão tal que tem a identidade tal"
		- Como usar os métodos de busca do document:
		```JavaScript
			document.getElementById('id-do-elemento') /*busca um elemento pelo seu id */
			document.querySelector('.classe-do-elemento') /*usa seletores CSS para encontrar o PRIMEIRO elemento que corresponde a regra*/
			document.querySelectorAll('.classe-dos-elementos') /*REtorna uma lista com TODOS os elementos correspondentes*/
		```
- getElement: HTML
- querySelector: CSS
- .innerHTML: lê e substitui tudo que tem em um elemento (apaga o que estava lá e substitui)
- .style: é uma ponte direta para o CSS é para aplicar estilos que mudam com a interação do usuário 
- .src .href: Para mudar uma imagem ou um link
- .addEventListener (responder a um evento)
