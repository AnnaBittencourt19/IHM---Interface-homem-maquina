## HTML
- **Sintaxe adequada:**
```html
 <!DOCTYPE html>: Tipo do documento é um HTML
 <html>: O que vai ter dentro do html
	 <head>: Cabeçalho 
		 <title> Título da página </title>
	 </head>
	 <body> Conteudo que realmente será exibido na página
		 <h1> título 1 </p>	 
		 <p> Paragrafo </p>
		 <ul>: Lista não ordenada
			 <li>elemento da lista</li>
		</ul>
		<a href= > LINK (o link deve ser "colado" após o href= </a>
		<img src= "caminho/da/imagem.png"> 
		<table> Tabela
			<tr> Linha 1 (tr, table row (linha coluna))
				<td> Coluna 1 da linha 1</td> 
				<td> Coluna 2 da linha 1</td>
			</tr>
			<tr> Linha 2
				<td> Coluna 1 da linha 2</td> 
				<td> Coluna 2 da linha 2</td>
			</tr>
			<div> Algum texto, alguma coisa 
			Div é para criar um "repatimento personalizavel"  
			</div>
		</table>
	 </body> 
 </html>
 
```
- HTML é aprender as tags e onde/como usa-las
